LeetCode 88. Merge Sorted Array 2019-06-07

题目描述

给定两个有序整数数组 nums1 和 nums2，将 nums2 合并到 nums1 中，使得 num1 成为一个有序数组。

说明:

初始化 nums1 和 nums2 的元素数量分别为 m 和 n。
你可以假设 nums1 有足够的空间（空间大小大于或等于 m + n）来保存 nums2 中的元素

tag

合并题 双指针

样例

算法1

(双指针 从后到前) O(n + m)
思路

三个指针分别指向 num1 最后一个有效元素，nums2最后一个元素，nums1 最后一个元素

复杂度分析

两个指针合起来一共移动两数组长度之和的距离。
