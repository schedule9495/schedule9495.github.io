LeetCode 371. Sum of Two Integers 2019-06-22

题目描述

不使用运算符 + 和 - ，计算两整数 a 、b之和。

tag

模拟加法题 模拟 位运算

样例

算法1

(位运算) O(1)
思路

在 Python 中，整数不是 32 位的，也就是说你一直循环左移并不会存在溢出的现象，这就需要我们手动对 Python 中的整数进行处理，手动模拟 32 位 INT 整型。

具体做法是将整数对 0x100000000 取模，保证该数从 32 位开始到最高位都是 0。
