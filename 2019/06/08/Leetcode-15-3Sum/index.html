Leetcode 15. 3Sum 2019-06-08

题目描述

给定一个包含 n 个整数的数组 nums，判断 nums 中是否存在三个元素 a，b，c ，使得 a + b + c = 0 ？找出所有满足条件且不重复的三元组。

注意：答案中不可以包含重复的三元组。

tag

算法1

(双指针) O(n^2)
思路

枚举第一个数，然后后面两个元素用双指针分别指向剩余元素开头和结尾。
