LeetCode 231. Power of Two 2019-06-08

题目描述

给定一个整数，编写一个函数来判断它是否是 2 的幂次方。

tag

位运算

样例

算法

(位运算) O(1)
思路

如果 n <= 0 则显然是 false
判断 n & (n - 1) == 0
2的幂次二进制表达中只有最高位为1，其余为0
例如8（1000）
8 - 1 = 7（0111）
1000 & 0111 = 0

算法2

(位运算) O(1)
思路

判断 (n & (-n)) == n ，取负运算是将 n 的二进制位取反然后再加 1。这种运算有个名称叫 lowbit，即取出 n 二进制位中从低位数第一个 1 的位置 k ，并返回 2^k。判断 (n & (-n)) == n 成立即意味着 n 二进制位中从低位数第一个 1 的位置就是最高位。
