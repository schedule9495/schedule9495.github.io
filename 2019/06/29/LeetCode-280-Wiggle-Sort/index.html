LeetCode 280. Wiggle Sort 2019-06-29

题目描述

给你一个无序的数组 nums, 将该数字 原地 重排后使得 nums[0] <= nums[1] >= nums[2] <= nums[3]…。

tag

摆动题

样例

算法1

(线性扫描) O(n)
思路

只在奇数索引处进行判断：
如果比左右两边的小,则与左右两边的交换

正确性验证：
如果 i 是当前遍历到的，那么 i-1 和 i+1 是需要判断的。
如果有发生交换，那么 i-1 处的值肯定比原先的小，那么肯定不会改变之前正确处理的顺序，即 i-1 肯定小于 i-2。
我们每次都保证了 i 与 i-1 的正确性，那么遍历到最后，整个数组都是正确的。
