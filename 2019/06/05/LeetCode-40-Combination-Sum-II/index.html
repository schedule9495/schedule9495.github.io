LeetCode 40. Combination Sum II 2019-06-05

题目描述

给定一个数组 candidates 和一个目标数 target ，找出 candidates 中所有可以使数字和为 target 的组合。

candidates 中的每个数字在每个组合中只能使用一次。

tag

组合题 DFS 回溯

样例

算法1

(DFS)
思路

sum(path) > target可剪枝，与组合总和的区别在于，每个数只可选一次，而组合总和能取无数次，所以是_dfs(i + 1, path)，而不是_dfs(i, path)，由于存在重复元素，所以需要排序与处理，遇到重复，排序预处理很重要。然后就可以用if i > index and candidates[i] == candidates[i - 1]来去重，这里i > index即可保证 candidates[i - 1] 存在，i > index， 说明 i 不会是第一个数，所以就可以有前一个数。
