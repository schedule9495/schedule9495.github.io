LeetCode 923. 3Sum With Multiplicity 2019-06-14

题目描述

给定一个整数数组 A，以及一个整数 target 作为目标值，返回满足 i < j < k 且 A[i] + A[j] + A[k] == target 的元组 i, j, k 的数量。

由于结果会非常大，请返回 结果除以 10^9 + 7 的余数。

3 <= A.length <= 3000
0 <= A[i] <= 100
0 <= target <= 300

(双指针) O(n^2)
思路

复杂度分析：

枚举第一个数 O(n)，后面两数双指针枚举也是 O(n)，总体 O(n ^ 2)。

算法2

(枚举) O(n + w^2)
思路

从小到大枚举 i 和 j，计算 k = target − i − j，满足 i <= j <=k。
根据i，j，k的大小情况，决定这组i，j，k究竟能有多少个。因为如果i == j，则枚举 j时必须去掉一个，结果是counter[i] counter[j - 1]，因为 i已经使用了一个。而i ！= j时则无须考虑这些，结果是 counter[i] counter[j]。

复杂度分析：

统计 counter 数组需要 O(n) 的时间，枚举数字需要 O(w^2) 的时间，故总时间复杂度为 O(n + w^2)。
