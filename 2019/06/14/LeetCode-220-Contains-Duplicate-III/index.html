LeetCode 220. Contains Duplicate III 2019-06-14

题目描述

给定一个整数数组，判断数组中是否有两个不同的索引 i 和 j，使得 nums [i] 和 nums [j] 的差的绝对值最大为 t，并且 i 和 j 之间的差的绝对值最大为 ķ。

tag

哈希表 分块

样例

算法1

(分块) O(n)
思路

将数组分成长度为t + 1的块，如果存在nums [i] 和 nums [j] 的差的绝对值最大为 t，则这两数映射到块中，必然在同一块中，或在相邻块中。检查即可。另注意 i 和 j 之间的差的绝对值最大为 ķ，所以要将j - i > k的映射删除。
