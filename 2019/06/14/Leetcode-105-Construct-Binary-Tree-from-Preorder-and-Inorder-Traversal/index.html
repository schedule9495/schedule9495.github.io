Leetcode 105. Construct Binary Tree from Preorder and Inorder Traversal 2019-06-14

题目描述

根据一棵树的前序遍历与中序遍历构造二叉树。

注意:
你可以假设树中没有重复的元素。

tag

二叉树重建题 二叉树

样例

算法1

(递归) O(n)
思路

对一棵树中序遍历和前序遍历所得的列表长度应该是一样长的。
可用哈希表加速查找前序遍历第一个节点在中序遍历中的下标。
