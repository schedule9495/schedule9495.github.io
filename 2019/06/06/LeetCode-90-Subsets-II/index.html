LeetCode 90. Subsets II 2019-06-06

题目描述

给定一个整数数组，可能包含重复元素。请返回它的所有子集（幂集）。

注意；答案中不能包含相同子集。

tag

子集 DFS 回溯

样例

算法1

(DFS) O(2^n * n)
思路

需要注意去重，先排序，后用 if i > index and nums[i] == nums[i - 1] 去重，i > index，说明 i 至少不是第一个元素，也就存在前一个元素。

复杂度分析

不同子集的个数最多有 2^n 个，另外存储答案时还需要 O(n)的计算量，所以时间复杂度是 O(n^2 * n)。
