LeetCode 62. Unique Paths 2019-06-23

题目描述

一个机器人位于一个 m x n 网格的左上角 （起始点在下图中标记为“Start” ）。

机器人每次只能向下或者向右移动一步。机器人试图达到网格的右下角（在下图中标记为“Finish”）。

问总共有多少条不同的路径？

说明：m 和 n 的值均不超过 100。

(数学) O(n)
思路

从网格左上角到网格右下角一共要走 m + n - 2步，在这些步数中，一共会向下走m - 1步，向右走n - 1步，因此就是经典的组合问题，结果就是C n−1 m+n−2。

空间复杂度为O(1)，时间复杂度为O(n)。
